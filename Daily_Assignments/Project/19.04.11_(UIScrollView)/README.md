<h3> 19.04.11 과제 </h3>

UIScrollView 에 UIPageControl을 사용하여 현재 페이지를 표시하는 화면 만들기 (별도 이미지 참고)

힌트:

- UIScrollView: Delegate 이용(scroll 관련),  pagingEnabled 속성 참고
- UIPageControl: currentPage 속성 및 numberOfPages 속성 참고
